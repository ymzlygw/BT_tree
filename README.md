# BT_tree
An example of Behavior Tree using Python 


## シナリオ
* 出発          
* 自分のHPが100以上
* 敵に寄る
* 友達Aと友達Bを同時に呼ぶ
* skill 1,2げランダムで２回発動
* 敵が死んだら、End1
* 敵が生きたら、 End2








## 設定
* 自分のHP　150
* 敵HP　120
* skill2発動確率100%
* skill1発動したら、敵HP50を減らす
* skill2を発動したら、敵HPを60を減らす

* input
     * skill1発動確率

* output
* 各行動結果をprint
